# Scripts Directory

This directory contains utility scripts and tools that are not meant to be displayed on the website.

## Contents

- `quiz.html` - Minimal HTML loader for the Hybrid Role Self-Assessment Quiz
- `quiz-generator.js` - Client-side JavaScript that dynamically generates the quiz interface

## Usage

The quiz can be accessed directly by opening `quiz.html` in a browser. It's a standalone, client-side application that requires no server-side processing.

## Hugo Configuration

This directory is ignored by Hugo via the `ignoreFiles` configuration in `config.toml`, ensuring these files are not processed or displayed on the website.
